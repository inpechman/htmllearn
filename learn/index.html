ברוכים הבאים לאתר ללימוד HTML

מה זה HTML?

HTML הוא שפת תגיות בשביל להציג דפי אינטרנט
בכל דף אינטרנט יש כמה איזורים, כמו כן ישנם סוגים שונים של תוכן שאותם נרצה להציג. בשביל שהאיזורים יהיו מסודרים וכן כדי שהתוכן יקבל מבנה מסויים, אנו משתמשים עם תגיות שונות.
לדוגמא:
עבור כותרת נשתמש באחד מהתגיות בין H1 ל H6
עבור פיסקה משתמשים עם תגית P
עבור טבלא התגית TABLE
וכדי לסדר אלמנטים לפי איזורים בדף נשתמש בתגית DIV

איך כותבים תגית?

שפת הבסיס לכתיבת התגיות הוא XML בו כל אלמנט מורכב משלושה חלקים - תגית פותחת, תגית סוגרת ותוכן האלמנט.
בתוכן האלמנט ניתן להכניס טקסט פשוט או להכניס אלמטים נוספים בשיטת הקינון (אפשר להכניס גם טקסט וגם אלמנטים במקביל)
כשמקננים אלמנטים אחד בתוך השני, יש לשים לב שהקינון יהיה מושלם ז"א שהאלמנטים יהיו אחד בתוך השני ולא חצי בפנים חצי בחוץ

תגיות לדוגמא:

כותרת: 

        <h1> זה כותרת מסוג H1 בשביל כותרת ראשית</h1>
        <h2> זה כותרת מסוג H2 בשביל כותרת משנית</h2>
        <h3> זה כותרת מסוג H3 </h3>
        <h4> זה כותרת מסוג H4,</h4>
        <h5> זה כותרת מסוג H5,</h5>
        <h6> זה כותרת מסוג H6 הקטן ביותר</h6>

תוצאה:

זה כותרת מסוג H1 בשביל כותרת ראשית

זה כותרת מסוג H2 בשביל כותרת משנית

זה כותרת מסוג H3

זה כותרת מסוג H4

זה כותרת מסוג H5
זה כותרת מסוג H6 הקטן ביותר

פיסקה:

<p>זה אלמנט מסוג P עבור פיסקאות</p>

תוצאה:

זה אלמנט מסוג P עבור פיסקאות

קישור:

<a href="https://google.com">קישור זה יעביר לדף החיפוש של גוגל</a>

תוצאה:

קישור זה יעביר לדף החיפוש של גוגל

שם הדף:

תגית מסוג <title> שקובע מה הטקסט שיופיע בשם של הדף (על הכרטיסייה למעלה)
תגית זו חייבת להיות בתוך איזור ה <head> של המסמך
בדף שלנו זה כתוב כך:
<title>learn HTML home page</title>

עיצוב טקסט בסיסי:

ניתן לתת עיצוב בסיסי כמו הדגשה וכו' ע"י תגיות
להלן מספר דוגמאות

הדגשה:

<p><b>זהו טקסט מודגש</b>
<strong>תגית נוספת לטקסט מודגש</strong></p>

תוצאה:

זהו טקסט מודגש
תגית נוספת לטקסט מודגש

הטייה:

<p><i>זהו טקסט מוטה</i>
<em>תגית נוספת לטקסט מוטה</em></p>

תוצאה:

זהו טקסט מוטה
תגית נוספת לטקסט מוטה

קו תחתון:

<p><u>זהו טקסט עם קו תחתון</u>
<ins>תגית נוספת לקו תחתון</ins> </p>

תוצאה:

זהו טקסט עם קו תחתון
תגית נוספת לקו תחתון

מירכוז טקסט:

<center>זהו טקסט ממורכז</center>

תוצאה:

זהו טקסט ממורכז

ציטוט:

<p><q>מוסיף סימני ציטוט (גרשיים) לטקסט</q></p>

תוצאה:

מוסיף סימני ציטוט (גרשיים) לטקסט

קוד:

מציג את הטקסט בסגנון קוד (פונט monospace ) דוגמא:

        <code dir="ltr">
            if (A == null) {
            return A
            }
        </code>

תוצאה:

if (A == null) { return A }

טקסט מפורמט:

מציג את הטקסט בפורמט המקור כולל רווחים וירידת שורות, דוגמא:

        <pre dir="ltr">
            while (i &lt; 10) {
                print i;
                i++;
            }
        </pre>

תוצאה:

                while (i < 10) {
                    print i;
                    i++;
                }

טקסט מחוק:

<del>מעביר סימן מחיקה על הטקסט (קו באמצע הטקסט)</del>

תוצאה:

מעביר סימן מחיקה על הטקסט (קו באמצע הטקסט)

טקסט קטן:

<small>זה טקסט בסגנון "קטן"</small>

תוצאה:

זה טקסט בסגנון "קטן"

תגיות להגדרת מבנה הדף:

ישנם מספר תגיות להגדרת מבנה הדף, חלקם מתפקדים כמו DIV מבחינה עיצובית ותפקידם סמנטלי בלבד
דגמא לתגיות כאלה:
התגית <nav></nav> בשביל תפריטים (אוסף לינקים משמעותיים)
התגית <footer></footer> בשביל תחתית הדף (אמור להכיל נתונים כלליים על הדף והאתר, כגון: זכויות יוצרים וכיו"ב
התגית <address></address> מיועד בשביל פרטים מזהים על הדף או על כתבה, בדרך כלל הטקסט מוטה

טבלאות:

ישנם כמה תגיות בטבלאות שיתוארו בדוגמא

ניתן לתת הגדרות שונות לעמודה שלמה ע"י שימוש בתגית COL ו COLGROUP 

<table>
        <colgroup>
        <col style="background-color: green">
        <col span="3" style=" background-color: red">
        </colgroup>
        <thead>
        <tr>
        <th>כותרת א</th>
        <th>כותרת ב</th>
        <th>כותרת ג</th>
        <th>כותרת ד</th>
        </tr>
        </thead>
        <tbody>
        <tr>
        <td>שורה 1 תא 1</td>
        <td>שורה 1 תא 2</td>
        <td>שורה 1 תא 3</td>
        <td>שורה 1 תא 4</td>
        </tr>
        <tr>
        <td>הבנתם</td>
        <td>את</td>
        <td>העיקרון</td>
        <td>אני</td>
        </tr>
        <tr>
        <td>ממלא</td>
        <td>בשביל</td>
        <td>הדוגמה</td>
        <td>בלבד</td>
        </tr>
        </tbody>
        </table>

תוצאה:

כותרת א כותרת ב כותרת ג כותרת ד
שורה 1 תא 1 שורה 1 תא 2 שורה 1 תא 3 שורה 1 תא 4
הבנתם את העיקרון אני
ממלא בשביל הדוגמה בלבד
